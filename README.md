# Inventory Platform 🏢

## Getting Started 🕛

##Prerequisites
    
    Composer
    
    Larvel
    
    SQLite / MySQL / PostgreSQL

## Built With 🎖️

    Laravel - The PHP framework used

    MySQL - The database management system used

## Installing 🏔️

    composer install

    php artisan key:generate

    php artisan migrate --seed

    php artisan serve
